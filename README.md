SDK for A20 devices (Cortex-A7 for FreeBSD)
===========================================


A simple development environment for ARMv7 boards based on Allwinners A20/H3/A83T/... processor. It provides basic component like compiler, env scripts (to set some environment variables like ${ARMHF_FBSD_HOME}) and more. Additional you find all infos and binarys/tools to setup one of the supported devices (see below). To make life easier you can use the provided scripts to clone useful external repositories like u-boot or build a kernel for your device. To make a ready to use sd-card you can use a dialog based script which guide you through the process.

Note: this is work in progess
